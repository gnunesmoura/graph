Detailed Description

template<class T, std::size_t N>
class gnm::Vertex< T, N >

Vertex of a graph with N adjacency lists.

Defines the behavior of the class Vertex when it have more than one type of adjacency.

Exemple of usage: If you have to calculate the the maximum flow of three diferent things in your system, instead of having three different graphs you can use three adjacency lists on every vertex.

Template Parameters
TType of the value stored in the Vertex.
NNumber of diferent kind of adjacencys the vertex may have.
